# Task Manager App

## 🚀 Features
- Create task
- View tasks
- Mark as completed
- Delete task

## 🛠 Tech Stack
- Backend: Spring Boot
- Frontend: HTML, JavaScript

## ▶️ How to Run

### Backend
1. Open project in IDE
2. Run Spring Boot application
3. Server starts at http://localhost:8080

### Frontend
1. Open index.html in browser

## 📌 API Endpoints
- GET /tasks
- POST /tasks
- PATCH /tasks/{id}
- DELETE /tasks/{id}

## 📌 Assumptions
- In-memory storage used
- No authentication
- Focus on functionality
